# Семейный Граф (Family Graph Service)

Полный стек приложения для построения и визуализации семейных деревьев с использованием Neo4j, NestJS (GraphQL) и React.

## Быстрый старт

### Предварительные требования

- Node.js 18+
- npm или yarn
- Docker и Docker Compose
